# Chat App
Chat app made with MERN Stack

## SetUp

1. Clone the repository:
   ```
   git clone https://github.com/Yaswanthkumar1100/ChatApp.git
   cd ChatApp
   ```
2. Setup .env file
    ```
    PORT=...
    MONGO_DB_URI=...
    JWT_SECRET=...
    NODE_ENV=...
    ```
## Build 

```
npm run build
```
## Run
```
npm start
```
